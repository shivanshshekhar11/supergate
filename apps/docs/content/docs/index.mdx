Supergate sits in front of OpenAI, Anthropic, Google, Cohere, and Mistral and gives your team one stable API with the operational features production systems need.

## What it does

| Feature | Description |
|---|---|
| **Unified API** | One OpenAI-compatible endpoint regardless of the underlying provider |
| **Multi-tenancy** | Isolated API keys, RBAC, and usage data per tenant |
| **Semantic caching** | pgvector cosine similarity cache — near-duplicate prompts served from cache |
| **Rate limiting** | Redis-backed sliding window limiter (TPM + RPM) per tenant |
| **Cost attribution** | Per-tenant token tracking with dollar-cost attribution |
| **BYOK** | Bring your own provider keys — AES-256-GCM encrypted at rest |
| **PII masking** | Regex scrub on request bodies before storage |
| **Observability** | Structured logs, circuit breaker states, cache hit rates |

## Architecture

```
Request → Auth → Rate Limit → PII Mask → Semantic Cache → Provider → Usage Logger
```

Every request flows through an ordered middleware pipeline. Adding a new concern is another hook — not a rewrite.
