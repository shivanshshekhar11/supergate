## Circuit breakers

Each provider has an independent circuit breaker with three states:

| State | Behavior |
|---|---|
| `CLOSED` | Normal operation |
| `OPEN` | Requests rejected immediately (503) |
| `HALF_OPEN` | One probe request allowed — success closes, failure re-opens |

The circuit opens after 5 consecutive failures and resets after 30 seconds. Check current states at `GET /health`.
